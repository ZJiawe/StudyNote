# 跳转

项目之间跳转

```
Ctrl+Alt+ ]
```

展示快捷键

```
ctrl + shilft + A
```

展现最近文件

```
ctrl + E
```

跳转上一次编辑的地方

```
Ctrl+Shift +backgrround
```

跳转到编辑框里

```
esc
```

跳转文件树

```
Alt+1
```

确定书签

```
F11    //跳转书签   shift + F11
```

浏览位置

```
Ctrl + Alt + 左/右
```
