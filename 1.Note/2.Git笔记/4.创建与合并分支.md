#### 1.创建并切换分支

```
$ git checkout -b dev
```

#### 2.查看当前分支

```
$ git branch
```

#### 3.创建分支

```
git branch <name>
```

#### 4.合并某分支到当前分支

```
git merge <name>
```

```
$ git merge dev  //dev分支的工作成果合到master
```

#### 5.删除分支

```
git branch -d <name>
```

#### 6.切换分支

```
git checkout <name>
```

```
$ git checkout master //切换到master
```
